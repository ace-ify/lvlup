# Daily Level-Up Tasks (8 Weeks)

Use this as your start/end sheet every day.

## Course Context (fixed)
- ML tasks come from your Udemy ML/NLP/MLOps course (filtered path only).
- Agentic tasks come from your Udemy Agentic AI (LangGraph/LangChain) course.
- For daily check-ins, include course section name completed.

## How We Will Use This Daily
- Start message to me: `W{week} D{day} start`
- End message to me: `W{week} D{day} done` + what is unchecked
- You only do the tasks listed for that day. No extra switching.

## Daily Stop Rule (anti-overload)
- Hard stop after 3 deep blocks + DSA + log.
- If any deep block is unfinished, move it to next day Top-1.
- Do not add new topics after Block 3.

## Fixed Daily Blocks
- Block A (60m): Backend or Agentic AI (as assigned)
- Block B (60m): Backend or Agentic AI (as assigned)
- DSA (45m): 2 questions (or revision day rules)
- ML (Deep day 60m / Light day 30m)
- Log (10m): built, stuck, next-first-task

## Weekly Context Pattern (same every week)
- Day 1 (Mon): Backend-heavy + ML light
- Day 2 (Tue): Agentic-heavy + ML deep
- Day 3 (Wed): Backend-heavy + ML light
- Day 4 (Thu): Agentic-heavy + ML deep
- Day 5 (Fri): Integration-heavy + ML light
- Day 6 (Sat): Buffer + revision + backlog clear
- Day 7 (Sun): Scorecard + next week setup
